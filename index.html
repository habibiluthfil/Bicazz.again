Tentang Bicazz

Bicazz adalah aplikasi yang berkaitan dengan keuangan. Aplikasi ini dapat membantu pengguna dalam alur keluar masuknya kas (uang) pengguna dengan fitur pencatatan, target tabungan dan split bill.

"Di tengah meningkatnya gaya konsumsi dan gaya hidup digital, banyak orang — terutama generasi muda mengalami kesulitan mengatur keuangan. Bicazz hadir sebagai penolong setia Anda."

❓ Kenapa Bicazz dibuat?

  • 🔴 Banyak pelajar tidak mencatat keuangan secara rutin
  • 🔴 Sulit menyisihkan uang untuk menabung
  • 🔴 Sering bingung menghitung rincian saat makan bersama

⭐ Kelebihan Kami

  • 🟢 Antarmuka yang intuitif dan mudah digunakan
  • 🟢 Semua fitur finansial dalam satu aplikasi praktis
  • 🟢 Visualisasi data dengan grafik yang interaktif

Dokumentasi Proses

Perjalanan Kelompok THE-lNCREDIBLES dalam membangun Bicazz dari nol.

1. Tahap Ideasi & Brainstorming

Kami menganalisis masalah umum mahasiswa: gaya hidup konsumtif dan kesulitan menabung. Dari sini, tercetus ide "Bicazz" (Biaya & Kas), terinspirasi dari kebutuhan akan "pahlawan super" keuangan (The Incredibles).

2. Desain UI/UX (Prototyping)

Membuat wireframe dan desain awal menggunakan Canva. Memilih palet warna khas Incredibles (Merah, Oranye, Kuning, Hitam) dan gaya *soft rounded* agar aplikasi terasa ramah dan tidak kaku.

3. Pengembangan (Coding)

Mengimplementasikan desain ke dalam kode menggunakan HTML, CSS (Tailwind CSS untuk *styling* cepat dan responsif), serta JavaScript untuk logika aplikasi (Kalkulasi Kas, Target Tabungan, Split Bill).

4. Pengujian & Penyempurnaan (Testing)

Melakukan *bug fixing*, memastikan grafik dari `Chart.js` merender dengan benar, dan merombak UI produk menjadi sistem *swipe card* yang modern dan futuristik.
